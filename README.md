
# 📦 TAEBAEKFOOD 스마트 주문관리 시스템 (로컬 가이드)

본 어플리케이션은 최신 웹 기술로 제작되어 사용자의 컴퓨터에서 직접 실행하고 편집할 수 있습니다.

## 🚀 PC에서 실행하는 방법

1.  **Node.js 설치**: [nodejs.org](https://nodejs.org/)에서 LTS 버전을 설치하세요.
2.  **프로젝트 폴더 생성**: 원하는 위치에 폴더를 만들고 제가 제공한 모든 파일을 해당 폴더 안에 복사해 넣으세요.
3.  **라이브러리 설치**: 터미널(CMD 또는 PowerShell)을 열고 해당 폴더로 이동한 뒤 아래 명령어를 입력하세요.
    ```bash
    npm install
    ```
4.  **서버 실행**: 설치가 완료되면 아래 명령어를 입력하세요.
    ```bash
    npm run dev
    ```
5.  **접속**: 브라우저에서 `http://localhost:3000`으로 접속하면 어플이 실행됩니다.

## 🛠️ 편집 방법

1.  **VS Code 설치**: 코드 편집을 위해 [Visual Studio Code](https://code.visualstudio.com/)를 권장합니다.
2.  **파일 수정**: `App.tsx`나 `components/` 폴더 안의 파일들을 수정하고 저장하면, 실행 중인 웹페이지에 즉시 반영됩니다.
3.  **AI 기능 설정**: 로컬 실행 시 Gemini AI를 사용하려면 폴더 루트에 `.env` 파일을 만들고 아래 내용을 넣으세요.
    ```env
    VITE_API_KEY=여러분의_구글_제미나이_키
    ```

## 📱 데스크탑 앱으로 설치하기

1.  웹 브라우저(Chrome/Edge)로 접속합니다.
2.  주소창 우측의 **[앱 설치]** 아이콘을 클릭합니다.
3.  이제 바탕화면 아이콘을 통해 브라우저 없이 독립된 프로그램처럼 사용할 수 있습니다.
